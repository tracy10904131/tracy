NCT


NCT取自「Neo Culture Technology（新文化技術）」的首字母，象徵著NCT是一組以全世界為舞台，具有開放性個擴張性，成員無上限的組合，目前有三個固定小分隊


NCT 127

                  

泰一  1994年6月14日

泰可愛 : 出自NCT LIFE團結大會，節目組上的字幕，因為做甚麼都很可愛還跳了可愛操。

其他 : 飯圈還會用"一葛格"、"一哥"都是在說泰一。

                  

Johnny 1995年2月9日

囧尼 : Johnny的中文諧音

                  

泰容 1995年7月1日

TY Track : 出道前出演過RV前輩的Be Natural，其中一句Rap詞是"TY Track TY Track"，那陣子成員們都會這樣叫他。

TY : TaeYong的縮寫。

絨絨 : 飯圈用暱稱，容容的可愛版本選字。

                  

悠太 1995年10月26日

章魚小丸子 : 因為大阪出產章魚燒很有名，而悠太又是日本人，出道初期粉絲也常叫他丸子。

其他 :Healing Smile  山男人  中本桑、悠太桑

                  

道英 1996年2月1日

豆英 : 名字諧音

雪貂、兔子 : 本人不認為像但是成員們都認證過道英長的像兔子，本人為了粉絲勉強接受這個暱稱。

貂老師 : 貂為長相相似，老師的部分是因為道英對粉絲很嚴格，簽售不接受撩還很會用人生道理教育粉絲，還會挑剔我們應援不夠整齊等等雜念。

                  

在玹 1997年2月14日

情人節男孩 : 2/14情人節出生因此被稱為Valentine Boy

學長 : 來自有名的"新東中學學長事件"，在電台收到一封告白信念到最後一句才發現主角就是自己，而且那位學妹本人還坐在下面，因此後來粉絲都叫他學長。

                  

Winwin1997年10月28日

昀昀 : 是Winwin的中文諧音，本人自我介紹認證。算是官方用名了。

                  

廷祐 1998年2月19日

小九 : 錕哥有跟廷祐說過粉絲會叫他小九，因為在廷祐出道前Rookies每個人都有一個編號(印在衣服上)，分別從1編到14號(泰容是1-志晟是14)，但是這個編號卻沒有9，明明當時只有13個人卻編到14號，因此大家認為有一個神秘的九號練習生，再加上飯拍等等粉絲就認為廷祐就是當時那個九號，因此叫他小九。

隱真瘋 : 直播裡成員們認證，本人有些疑惑還不太確定自己是，意思是看起來很正常實則是個瘋子的意思。
